# Masakali Golden Garden Redesign

Static product page redesign for the Masakali.Co Golden Garden Maxi Dress.

## Files

- `index.html` is the GitHub Pages entry point.
- `outputs/masakali-golden-garden-redesign.html` is the original deliverable copy.

## GitHub Pages

Publish from the repository root on the `main` branch. The page will load directly from `index.html`.
